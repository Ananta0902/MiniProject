# ❤️ Cardiovascular Disease Risk Prediction using Machine Learning

A machine learning project that predicts the likelihood of cardiovascular disease using patient health records. The project evaluates multiple supervised learning algorithms to identify the most accurate model for early disease detection and healthcare decision support.

> 🎓 Academic Team Project at Indian Institute of Engineering Science and Technology (IIEST), Shibpur as  a part of coursework.

---

## 📌 Overview

Cardiovascular diseases are one of the leading causes of mortality worldwide. Early prediction can assist healthcare professionals in timely diagnosis and preventive treatment.

This project analyzes patient clinical data, performs exploratory data analysis (EDA), trains multiple machine learning classifiers, and compares their performance to determine the most accurate prediction model.

---

## 🚀 Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature correlation analysis
- Training multiple Machine Learning models
- Performance comparison across classifiers
- Cardiovascular disease risk prediction
